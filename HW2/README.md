# 選擇的題目
「財經新聞的語意分析及對股價的影響」
目的
* 希望了解新聞出現的字詞與往後個股漲跌的關係。
* 先搜集財經類新聞中的字詞，再依據往後的個股股價漲跌作為分類:
1. 可以將字詞本身做分類來推測。
2. 由於漲跌下字詞本身之間的共現性可能不同，可以藉此來推測往後漲跌的情形。
* 選取台積電作為標的
